# BaoGuaEncoder
更好用的字符编码转换工具

**特点：** 
- 完全本地运行，处理敏感数据无需担心外泄风险。
- 输入自动转输出，不用手动点击转换按钮，对于需要反复修改数据操作更加方便。
- 支持多级不同编码转换，不用来回粘贴输入输出。
- 自带一键转大小写、复制、粘贴功能菜单，操作更便捷。

**功能：**  
编码转换：
- URL
- Base64
- Base64（自动补全）
- Unicode
- ASCII
- Html转义
- Unix时间戳（秒）
- Unix时间戳（毫秒）

哈希计算：
- MD5
- CRC32
- CRC64
- SHA1
- SHA256
- SHA384
- SHA512
